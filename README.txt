Sie können die exportierte Datenbank 'datenspeicher.sql' verwenden.
Allerdings bietet unser usermenu auch die Möglichkeit DBs und Tabellen zu erstellen. 
Dieses Programm ist nicht an eine bestimmte Datenbank geknüpft.
Sie können das usermenu und das Monitoring-Programm auf jede beliebige DB anwenden.
